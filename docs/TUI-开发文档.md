# Codex Session Viewer - 交互式终端工具

## 概述

一个跨平台的交互式终端工具，用于查看、搜索和导出 OpenAI Codex CLI 的历史 session 记录。

## 核心理念

**单一入口，交互操作**

- 运行一个命令直接进入交互式界面
- 所有功能都在 TUI 中完成
- 通过键盘快捷键操作

## 键盘快捷键

### 全局

| 快捷键 | 功能 |
|--------|------|
| `q` | 退出程序 |
| `Ctrl+C` | 强制退出 |

### Session 列表

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 导航列表 |
| `Enter` | 查看选中的 session |
| `/` | 打开搜索 |
| `e` | 导出选中的 session |
| `r` | 刷新列表 |

### Session 详情

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 滚动内容 |
| `Esc` | 返回列表 |
| `/` | 在当前 session 中搜索 |
| `e` | 导出当前 session |

### 搜索

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 执行搜索 |
| `Esc` | 取消搜索 |
| `r` | 切换正则模式 |
| `t` | 切换消息类型过滤 |

### 导出

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 选择格式 |
| `Enter` | 确认导出 |
| `Esc` | 取消导出 |

## Session 文件格式

### 存储位置

```
~/.codex/sessions/
├── 2026/
│   ├── 04/
│   │   ├── 02/
│   │   │   ├── rollout-2026-04-02T10-00-10-uuid.jsonl
│   │   │   └── rollout-2026-04-02T15-54-55-uuid.jsonl
│   │   └── 03/
│   └── 03/
└── 2025/
```

### 文件格式

JSONL（每行一个 JSON 对象）：

```json
{"timestamp":"2026-04-02T10:00:00Z","type":"event_msg","payload":{"type":"user_message","message":"如何迁移数据库？"}}
{"timestamp":"2026-04-02T10:00:05Z","type":"event_msg","payload":{"type":"agent_message","message":"我来帮您查看..."}}
{"timestamp":"2026-04-02T10:00:10Z","type":"response_item","payload":{"type":"function_call","name":"shell_command","arguments":"{\"command\":\"ls\"}"}}
```
